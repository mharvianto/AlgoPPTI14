# Session 01

## Type Data

| Type Data | Description | Format |
| --------- | ----------- | :----: |
| `short` | Angka bulat | `%d` |
| `int` | Angka bulat | `%d` |
| `long` | Angka bulat | `%ld` |
| `char` | Huruf/character | `%c` | 
| `char[]` | string/array of char/word | `%s` |
| `float` | Angka desimal | `%f` |
| `double` | Angka desimal | `%lf` | 


## Variabel
[type data] [nama variable]

Ex:
```c++
int angka;
```

```c++
char nama[50];
```
